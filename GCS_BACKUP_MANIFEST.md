# GCS backups of model weights (adapters + FFT full models)

Where weights from the cat/Qwen2.5-7B subliminal-learning experiments live on GCS
after being offloaded from the `/data/user_data/lawrencf` quota (501G, chronically
near-full). **Logs (`summary.json`/`progress_log.json`/`loss_log.json`) always stay
local under `…/results/<run>/`; only the heavy weight files are moved to GCS.**

GCS bucket root: `gs://lawrencf-persona-system/persona-system/lora_artifact_cat_qwen7b/`

## Layout

| what | local (pre-offload) | GCS location |
|---|---|---|
| LoRA adapters (all waves) | `…/adapters/<run>/` | `…/adapters/<run>/` |
| FFT full models | `…/fft_full/<run>_full/` or `…/results/<run>/` | `…/fft_weights/<run>/` |

Each adapter dir holds `adapter_model.safetensors` + `adapter_config.json` +
tokenizer/chat-template. Each FFT model dir holds `model.safetensors` (~15.2G) +
config/tokenizer.

## Machine-readable manifest

`gcs_adapter_manifest.tsv` (repo root) — one row per offloaded adapter:
`timestamp  run_name  size_bytes  gcs_uri  status` (status `OK` = uploaded,
byte-verified, local deleted). Generated by the scripts below.

## Scripts (all byte-verify `adapter_model.safetensors`/`model.safetensors` size on
GCS before deleting the local copy; idempotent)

- `offload_xl500k_adapters.sh` — current 500k rank sweep adapters (`cat7b_xl500k_*`),
  gated on the run's `summary.json` (only finished runs).
- `offload_loop_xl500k.sh` — runs the above every 15 min until the sweep drains.
- `backup_old_adapters.sh` — all pre-sweep adapters (`cat7b_x26_*`,
  `cat7b_xl{2x,4x,8x,8x1ep}_*`).

## To restore a model

`gsutil -m cp -r gs://…/adapters/<run>/ ./local_dir/` (adapter) or
`gsutil -m cp -r gs://…/fft_weights/<run>/ ./local_dir/` (full model), then load
with PEFT (`PeftModel.from_pretrained`) or `AutoModelForCausalLM.from_pretrained`.
